CSS Temel Ödevi

Sadece HTML5 + CSS3 kullanın.

İçerik

Bu alanda örnek içerik metinleri bulunmaktadır. Css özellikleriyle bu metinler arasında boşluklar (line-height) ve hizalamalar (text-align) düzenleyiniz.

Önce CSS ile kutuları yan yana getirmeyi, ardından alt alta yerleştirmeyi öğrendik. Şimdi ise CSS seçicilerini kullanarak grup ve hiyerarşik seçimler yapmayı göreceğiz.

Yan Menü / Bilgi

Bu alan yan menü veya bilgi kısmıdır. CSS ile genişlik (width), yükseklik (height), kenar boşlukları (margin/padding) ve kenarlık (border) özelliklerini kullanarak düzenleyebiliriz.

Öğrendiklerimizi hiyerarşik ve grup seçicilerle uygulayıp farklı bölümlere stil verebiliriz.
